# 视频背景去除工具 - MVP 需求文档

## 项目概述

**产品名称**: VideoMatte (暂定)  
**定位**: 在线 AI 视频背景去除工具  
**目标用户**: 短视频创作者、电商卖家、自媒体从业者  
**核心价值**: 无需绿幕，AI 自动抠像，30 秒完成视频背景去除

---

## MVP 核心功能

### 1. 视频上传
- 支持格式: MP4, MOV, WEBM
- 文件大小限制: 50MB (约 30 秒 1080p 视频)
- 分辨率限制: 最高 1080p
- 拖拽上传 + 点击选择

### 2. AI 处理
- 调用 Replicate RVM API
- 自动去除背景
- 生成透明背景视频 (WebM with alpha)
- 处理时间: 预计 1-3 分钟

### 3. 结果预览与下载
- 在线预览处理结果
- 提供两种下载格式:
  - 透明背景 WebM (用于合成)
  - 绿幕背景 MP4 (兼容性更好)
- 免费版带水印

### 4. 简单落地页
- Hero 区: 产品介绍 + 示例对比
- 上传区: 拖拽上传界面
- 功能说明: 3 个使用场景 (短视频/电商/广告)
- FAQ: 5-8 个常见问题

## 技术方案

### 前端
**技术栈**: Next.js 14 + TypeScript + Tailwind CSS

**核心页面**:
- `/` - 首页 + 上传
- `/processing/[id]` - 处理进度页
- `/result/[id]` - 结果预览下载页

**关键功能**:
- 文件上传: `react-dropzone`
- 视频预览: 原生 `<video>` 标签
- 进度轮询: 每 3 秒查询一次状态

### 后端
**技术栈**: Next.js API Routes (Serverless)

**API 端点**:
```
POST /api/upload        - 上传视频，返回任务 ID
GET  /api/status/:id    - 查询处理状态
GET  /api/download/:id  - 获取下载链接
```

**处理流程**:
1. 接收上传 → 存储到 Cloudflare R2
2. 调用 Replicate API 处理
3. 轮询 Replicate 状态
4. 处理完成 → 下载结果到 R2
5. 返回下载链接

### AI 服务
**方案**: Replicate API  
**模型**: `chenxwh/rvm` (Robust Video Matting)  
**成本**: ~$0.02-0.05/视频

### 存储
**方案**: Cloudflare R2  
**存储策略**:
- 原始视频: 保留 24 小时
- 处理结果: 保留 7 天
- 自动清理过期文件

### 部署
- **前端**: Cloudflare Pages
- **后端**: Next.js API Routes (Vercel 或 Cloudflare Pages Functions)
- **域名**: videomatte.com (待注册)

## 用户流程

### 主流程
```
1. 用户访问首页
   ↓
2. 拖拽/选择视频上传 (最大 50MB)
   ↓
3. 显示上传进度条
   ↓
4. 跳转到处理页面，显示预计等待时间
   ↓
5. 轮询处理状态 (每 3 秒)
   ↓
6. 处理完成，跳转到结果页
   ↓
7. 预览对比效果 (原视频 vs 去背景)
   ↓
8. 下载结果 (WebM 透明 / MP4 绿幕)
```

### 异常流程
- 上传失败 → 提示重试
- 处理超时 (>5 分钟) → 提示联系客服
- 文件过大 → 提示升级 Pro 版

## 技术风险与应对

### 风险 1: Replicate API 成本过高
**应对**: 
- 设置每日处理上限
- 引导用户升级付费
- 后期自建 GPU 服务

### 风险 2: 处理速度慢，用户流失
**应对**:
- 优化视频预处理 (压缩、降分辨率)
- 显示准确的预计时间
- 提供邮件通知功能

### 风险 3: 存储成本增长
**应对**:
- 严格执行自动清理策略
- 引导用户及时下载
- 付费用户延长保存时间
